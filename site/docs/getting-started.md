# Getting Started

busydev is designed for developers running multiple AI coding sessions in parallel across branches and repositories.

## Install and run

```bash
npm install
cargo tauri dev
```

## First-time flow

1. Create a project in the sidebar using your local git repository.
2. Create one or more workspaces tied to branches or issue tickets.
3. Choose an agent adapter and send your first prompt.
4. Review changes in the diff panel and run commands in terminal tabs.

Continue to [Setup](./setup.md) for full local environment details.
