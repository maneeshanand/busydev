busydev
Docs GitHub

Coordinate coding agents without terminal chaos.

busydev is a desktop command center for Codex, Claude Code, and other CLI agents. Run parallel sessions, isolate worktrees, review diffs, and keep attention signals in one place.

Get Started Core Workflows

What you can do

Run multiple agent sessions

Track status and output across workspaces with one unified view.

Keep each task isolated

Use automatic git worktrees per workspace to avoid branch collisions.

Review changes quickly

Use built-in diff tools and terminal tabs without context switching.

Stay LLM-agnostic

Plug in supported adapters while keeping one consistent workflow.
